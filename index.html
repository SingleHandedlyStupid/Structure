Structure UI Library v1.0.5

UI Library

local UI = loadstring(game:HttpGet("https://raw.githubusercontent.com/SingleHandedlyStupid/Structure/refs/heads/main/Str7ct.lua"))()

Window

Creates a new window.


local win = UI:Window("Admin Panel")
win:BindToggle(Enum.KeyCode.RightShift)

Tab

Adds a tab inside a window.


local main = win:Tab("Main")
main:Button("Hello", function() print("Hello") end)
main:Slider("Speed", 16, 200, function(v) ... end)
main:Dropdown("Team", {"Red","Blue","Green"}, function(v) ... end)

Notifications

Displays a popup notification:


UI:Notify("Info", "Panel loaded", 3)
